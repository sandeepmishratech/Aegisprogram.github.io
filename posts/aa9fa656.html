我的实用工具(电脑篇)

Posted on 2021-08-27 Edited on 2021-08-28 In 应用 Views:
Symbols count in article: 5.4k Reading time ≈ 5 mins.


每个人在自己使用电脑的过程中一定有一些离不开的精品应用，我们习惯将其称之为“装机必备”。今天我就盘点一下自己电脑上那些装机必备软件。

实用工具类

BleachBit

GitHub地址：https://github.com/bleachbit/bleachbit

官网：https://www.bleachbit.org/

BleachBit是一款开源的系统清理工具，它可以释放磁盘空间，保护您的隐私，清除缓存，删除cookies、internet历史、临时文件、日志和丢弃的垃圾文件等，支持清除应用的残留数据，切碎文件防止恢复、隐藏被删除文件的痕迹等功能，仅适用于Windows和Linux。

Dism++

GitHub地址：https://github.com/Chuyu-Team/Dism-Multi-language

官网：https://www.chuyu.me/en/index.html

Dism++ 是一款 Windows 系统管理优化工具，它能帮我们解决安装、维护系统的一大痛点问题，自定义设置和优化。是一个 “一站式” 管理优化工具集，仅适用于Windows Vista/7/8/8.1/10

Listary

官网：https://www.listary.com/

Listary是一款用于Windows的文件搜索、定位的辅助软件。它提供了便捷、人性化的文件（夹）定位方式，改善Windows传统低效的文件打开/保存对话框，同时改善了常见文件管理器中文件夹切换的效率。
虽然软件本身分为专业版和免费版，但是对于一般人来说，免费版就已经足够使用。相比于大多数人熟知的everything，我更加青睐这款软件文件夹记忆功能，无论你在哪一个窗口，你都能很快的找到你你想要的文件夹。还能用过快捷键集成快捷启动应用在上面。

geek uninstaller

官网：https://geekuninstaller.com/

geek uninstaller是一款极为优秀的Windows应用卸载软件，轻巧、功能强大，易使用。它能够在卸载应用的同时彻底清理掉应用残留的注册表，还可以卸载Windows微软商店中你不想要的自带应用。软件不需要安装，解压即可用。

Bandzip

官网：https://www.bandisoft.com/bandizip/

Bandzip是一个功能强大的，简单易用的免费压缩软件，解压缩的效率都很高，支持的格式也很丰富。这是我最喜欢的压缩工具。配合HoneyView即可实现不解压预览压缩包内的图片。

ThunderBird

源代码地址：https://hg.mozilla.org/comm-central

官网：https://www.thunderbird.net/en-US/

Mozilla Thunderbird，非正式中文名称为雷鸟，是由Mozilla基金会研发的一款自由及开放源码的跨平台电邮客户端、新闻阅读器、聚合器以及即时通信软件。此软件默认安装于Ubuntu之上。 现在已经可适用于Windows，Linux和MacOS。

SuperF4

GitHub地址：https://github.com/stefansundin/superf4

官网：https://stefansundin.github.io/superf4/

这是一个仅适用于Windows平台的开源的强制杀死进程的应用。

Snipaste

官网：https://www.snipaste.com/

这是一款免费，简单，小巧的截图软件。功能强大，易于使用。不过，这款软件无法实现长截图功能，如果想要实现长截图的话，不妨试试这个FastStone Capture

效率工具

Powertoys

GitHub地址：https://github.com/microsoft/PowerToys

PowerToys是一组免费的系统工具软件，由微软为Windows操作系统上的系统管理员设计。这些程序为系统加入或变更了一些功能，并加入更多自定义选项以提高生产力。PowerToys可用于Windows 95、Windows XP和Windows 10。Windows 10版。用的好，绝对是一款神器。

Rubick

GitHub地址：https://github.com/clouDr-f2e/rubick

官网：https://cloudr-f2e.github.io/rubick/

基于 electron 的开源工具箱，自由集成丰富插件。 如果你是一个utools工具箱的重度依赖者，那么这款开源的替代应用对你来说也值得一试。

ScreenToGIF

GitHub地址：https://github.com/NickeManarin/ScreenToGif

官网：https://www.screentogif.com/

ScreenToGif 是一款录制屏幕、摄像头、画板于一身的 GIF 制作软件。

IDM

官网：https://www.internetdownloadmanager.com/

是一个用于Windows系统的下载管理器。 它功能强大，可以说是目前Windows平台好用，最省心的下载管理器。
我在上一篇的博文中也说到了这个下载工具。

开发工具

VSCode

GitHub地址：https://github.com/microsoft/vscode

官网：https://code.visualstudio.com/

根据Stackoverflow的统计，这是程序员最喜爱的IDE工具。微软良心巨作。

WindowsTerminal

GitHub地址：https://github.com/microsoft/terminal

Windows Terminal 顾名思义就是「Windows 终端」，是一款全新的，现代的快速、高效且功能强大的终端应用程序。它同时适用于会用到：「命令提示符」、PowerShell 和 WSL 等命令行交互的用户。是一款开源、免费、高效、现代、支持高度自定义的终端工具。关于WindowsTerminal相关介绍文章可以参考新生代 Windows 终端：Windows Terminal 的全面自定义或者参考我之前的博客。如果你觉得Windows Terminal的配置过于耗时，那么你可以选择下载自带Git的cmder，建议下载full版本的。

Atom

GitHub地址：https://github.com/atom/atom

官网：https://atom.io/

Atom是github专门为程序员推出的一个跨平台文本编辑器。具有简洁和直观的图形用户界面，并有很多有趣的特点：支持CSS，HTML，JavaScript等网页编程语言。它支持宏，自动完成分屏功能，集成了文件管理器。

不过由于是使用electron和JavaScript进行编写的，所以存在性能不足问题。

文本编辑器

Typora

官网：https://typora.io/

Typora 是一款支持实时预览的 Markdown 文本编辑器。它有 OS X、Windows、Linux 三个平台的版本，并且由于仍在测试中，是完全免费的。关于更多的信息可以参考Typora 完全使用详解

当然，Atom和VSCode可以通过安装相对应的扩展来实现Markdown写作预览效果，效果也是不错的。

播放器

VLC

源代码地址：https://code.videolan.org/explore/projects/starred

官网：https://www.videolan.org/vlc/

是一款开源的、跨平台的、可扩展的、多媒体播放器、流媒体服务器及框架，可播放大多数多媒体文件，以及DVD、音频CD、VCD及各类流媒体协议。VLC不仅仅是一个音/视频播放器，它也可以作为小型的视频服务器(或作为流媒体服务器使用)，更可以一边播放一边转码，把视频流发送到网络上。VLC最为突出的就是网络流的播放功能。

potplayer

官网：https://potplayer.daum.net/

potplayer是KMPlayer的原作者姜勇囍进入Daum公司后的新一代作品，可以说是Windows平台上最强的播放器，不仅支持众多的视频格式，而且颜值高，功能丰富，上手简单。

ZY-Player

GitHub地址：https://github.com/cuiocean/ZY-Player

官网：https://zyplayer.fun/

跨平台桌面端视频资源播放器.简洁无广告.免费高颜值。算是一个很良心的资源播放器，重在内置很多的播放源。

LX-Music

GitHub地址：https://github.com/lyswhut/lx-music-desktop

这是一个跨平台的音乐资源播放器，内置了国内几大主流的音乐源，基于 electron和JavaScript。如果想要 在线听多个平台的音乐的话，选它就没错。

YesPlayMusic

GitHub地址：https://github.com/qier222/YesPlayMusic

官网：https://music.qier222.com/#/

高颜值的第三方网易云播放器，支持 Windows / macOS / Linux 。如果你是一个网易云音乐的重度使用用户，但是又已经收够了网易云客户端的臃肿，那么这一款开源的第三方值得你一试。

阅读器

Fluent Reader

GitHub地址：https://github.com/yang991178/fluent-reader

官网：https://hyliu.me/fluent-reader/

基于Electron, React的现代化的桌面端RSS阅读器。同时还有移动客户端

Kindle

官网：https://amzn.to/3DpDit7

Amazon Kindle是亚马逊公司设计的一系列电子书阅读器。用户可以通过无线网络使用Kindle购买、下载和阅读电子书、报纸、杂志、部落格及其他电子媒体。是我目前最喜欢的电子阅读器。

Resilio Sync

官网：https://www.resilio.com/

Resilio Sync 是由BitTorrent公司开发的专有的对等网络数据同步工具，可在Windows、OS X、Linux、Android、iOS和FreeBSD上使用。其可在局域网、互联网上通过安全的、分布式的P2P技术在不同设备之间同步文件。

Resilio Sync最大的优势在于其不需要第三方服务器，几乎不受存储空间和流量限制。由于Resilio Sync是一款采用 BitTorrent 协议的P2P同步工具，传输速度基本上只受用户网络带宽的制约，它可以支持多人同时同步，而且，越多人对文件夹进行同步，则同步速度越快。它支持多平台，可以通过密钥进行文件分享。

由于Resilio Sync的数据不在云端，而是存储在本地。所以拥有文件的电脑需要保持在线，才能同步到其他电脑去。而且由于采用类似 BT 的协议，因此在某些禁用 BT 下载的场景中可能会受到影响。

最后

以上这些就是我在使用Windows系统时难以舍弃的应用，以后说不定还会遇到不错的应用，那么就到时候再补充吧！
