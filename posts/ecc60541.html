10款优秀的下载工具

Posted on 2021-08-18 In 应用 Views:
Symbols count in article: 3.4k Reading time ≈ 3 mins.

整理收集那些好用的下载工具，节约下载等待的时间。

Aria2

GitHub：https://github.com/aria2/aria2

官方网站：https://aria2.github.io/

使用教程：https://bit.ly/2VWYp5j

Persepolis

GitHub：https://github.com/persepolisdm/persepolis

官方网站：https://persepolisdm.github.io/

使用教程：https://bit.ly/3sqg2Gl

Persepolis 是一个下载管理器和一个用于aria2. 它是用 Python 编写的。Persepolis 是一个免费和开源软件的示例。它是为 GNU/Linux 发行版、BSD、macOS 和 Microsoft Windows 开发的。

NeatDownloadManager

官方网站：https://bit.ly/3ma6iyQ

使用教程：https://zhuanlan.zhihu.com/p/197828813

NDM，全称Neat Download Manager，软件本身并不开源，使用起来也比较容易上手，支持安装浏览器扩展。

Free Download Manager

开源地址：https://bit.ly/3kaQkC6

官方网站：https://www.freedownloadmanager.org/

使用教程：https://bit.ly/3gdesCE

一个强大的现代下载加速器和Windows、macOS、Android和Linux的下载加速器。

xdown

官方网站：https://xdown.org/

使用教程：https://bit.ly/37VMhUd

专业的文件下载与分享工具,免费无广告的idm / torrent 合成体。

qBittorrent

GitHub：https://github.com/qbittorrent/qBittorrent

官方网站：https://www.qbittorrent.org/

使用教程：https://baiyunju.cc/3352

这是一款最好用的BT种子下载工具，功能强大，界面美观，开源免费，还可以跨平台使用。不过国内使用可能效果不是特别好，但是可以添加代理。

Motrix

GitHub：https://github.com/agalwood/Motrix

官方网站：https://motrix.app/

使用教程：https://bit.ly/3mewA36

这是一款全能的下载工具，支持高速下载 HTTP、FTP、BT、磁力链、百度网盘等资源，使用它下载百度网盘的资源是一个不错的选择。

uGet Download Manager

GitHub：https://github.com/ugetdm

官方网站：https://ugetdm.com/

使用教程：https://bit.ly/3iT3DHM

uGet 是一个下载管理器，它结合了轻量级资源使用和非常强大的功能集。

Transmission

GitHub：https://github.com/transmission/transmission

官方网站：https://transmissionbt.com/

使用教程：https://bit.ly/3xYytmo

说明：https://bit.ly/3sqXQfN

这是一个更加适合运行在服务器上的BT种子下载工具，搭配软路由食用效果更棒棒！

Internet Download Manager（IDM）

官方网站：https://www.internetdownloadmanager.com/

使用说明：https://bit.ly/37UzdP6

这是一款老牌的下载工具了，也是我现在用的最多的下载工具，它是一款集成在浏览器扩展上的下载工具，几乎不需要你配置什么，傻瓜式的操作。不过该工具不支持BT种子下载，不支持MacOS和Linux，需要付费使用。
