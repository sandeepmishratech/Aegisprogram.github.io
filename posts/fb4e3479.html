zsh配置

在MacOS和Linux的默认shell基本上都是zshell(zsh)，这个zsh功能十分的强大，不过配置起来相对麻烦一些。不过GitHub上有一个开源的on-my-zsh，它能够帮助我们更快更好的对zsh进行配置。

在Windows安装配置zsh需要一些准备工作：
1.需要安装WSL并且完成相关的配置
2.电脑上需要git
3.需要在下载的Linux发行版上创建好账户
4.将WSL集成到WindowsTerminal中

接下来的操作都是在WindowsTerminal中进行，如果上面几步没有做好的话，那么可以参考我之前的博客。我的WSL中使用的Linux发行版是Ubuntu18.04

首先在Ubuntu中下载好zsh sudo apt install -y zsh

接着将shell切换成zsh clsh -s /bin/shell，重启终端，输入echo $SHELL输出/bin/zsh表示成功。

插件(plugins)

oh-my-zsh的插件生态非常丰富，大家根据自己的需要去安装就可以，这里列举几个个人觉得比较需要的。

incr是一款自动提示插件，功能非常强大。

zsh-syntax-highlighting是一款高亮可用命令的插件。

zsh-autosuggestions这是一个命令自动补全插件，当你输入命令的几个字母，它会自动根据历史输入进行自动补全。

注意：如果操作过程中出现_arguments:448: _vim_files: function definition file not found错误，请执行：rm -f ~/.zcompdump即可。
